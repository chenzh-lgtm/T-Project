一、项目概述
============
本项目是一个微信心理诊断小程序，提供多种性格测试和心理健康评估功能。项目分为三个主要部分：
1. 微信小程序前端 - 用户使用的移动端界面
2. 后端API服务 - 提供数据存储和业务逻辑
3. 管理后台 - 管理员管理测试数据和用户信息

2.2 架构说明
------------
- **微信小程序**：用户在微信内使用的前端界面，负责展示测试、收集答案、显示结果
- **后端API服务**：处理业务逻辑，与数据库交互，提供数据接口
- **数据库**：存储所有测试数据、用户信息和测试记录
- **管理后台**：管理员用来管理测试内容和查看统计数据

三、安全设计
===========

3.1 数据传输安全
----------------
- 所有网络请求使用 HTTPS 协议，防止数据被窃取或篡改
- 接口请求需要携带 Token 进行身份验证
- 敏感数据（如用户信息）在传输过程中加密处理

3.2 身份认证安全
----------------
- 用户登录使用微信官方授权机制，确保用户身份真实可靠
- 管理员登录使用账号密码 + 验证码双重验证
- Token 有效期设置为 2 小时，过期后需要重新登录
- 密码使用 BCrypt 加密存储，即使数据库泄露也无法直接获取密码

3.3 防止常见攻击
----------------
- **防止 SQL 注入**：使用参数化查询，不直接拼接 SQL 语句
- **防止 XSS 攻击**：对用户输入的内容进行转义处理，避免恶意脚本执行
- **防止越权访问**：每个接口都检查用户权限，确保用户只能访问自己的数据
- **防止暴力破解**：登录失败多次后暂时锁定账号

3.4 数据存储安全
----------------
- 数据库设置访问权限，只允许指定服务器访问
- 定期备份数据库，防止数据丢失
- 用户答题记录加密存储，保护用户隐私

四、核心业务流程
===============

4.1 用户测试流程
----------------
1. 用户打开小程序，自动获取微信用户信息登录
2. 用户在首页选择测试分类（性格测试/心理测试/趣味测试）
3. 选择具体的测试项目，查看测试说明
4. 开始答题，按顺序回答问题
5. 提交答案，系统计算得分并匹配结果模板
6. 显示测试结果和分析建议
7. 用户可以分享结果或保存到相册

4.2 管理员管理流程
------------------
1. 管理员登录管理后台
2. 在测试管理模块创建/编辑测试项目
3. 在题目管理模块添加测试题目和选项
4. 在结果管理模块配置不同分值对应的结果模板
5. 查看数据统计，了解用户测试情况

六、关键设计说明
===============

6.1 测试结果计算逻辑
-------------------
- 用户答题后，系统根据选项对应的分值计算总分
- 根据总分匹配预设的结果模板（如 0-20 分对应结果A，21-40分对应结果B）
- 结果模板由管理员在后台配置

6.2 数据同步机制
----------------
- 用户答题进度自动保存到本地存储
- 提交测试结果时同步到服务器
- 测试记录实时存储到数据库

6.3 扩展性设计
--------------
- 测试分类和测试项目支持动态添加，无需修改代码
- 题目类型支持单选、多选、量表等多种形式
- 结果模板支持灵活配置，适应不同测试需求

七、部署说明
============

7.1 小程序部署
--------------
- 在微信公众平台注册小程序账号
- 上传小程序代码，提交审核
- 审核通过后发布上线

7.2 后端部署
------------
- 服务器环境：Linux + Nginx + Java 21
- 数据库：MySQL 8.0+
- 使用 Docker 容器化部署，方便管理和扩展

7.3 管理后台部署
----------------
- 可部署在同一服务器或独立服务器
- 需要配置域名和 HTTPS 证书
