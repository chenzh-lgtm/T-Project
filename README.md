# 性格、心理测试微信小程序

## 项目简介

本项目是一个专门为**大专生心理健康**问题开发的微信小程序，提供多种专业的心理测评服务，帮助老师及时了解学生的心理状态，及时发现潜在的心理问题。
**微信小程序搜索：【A性格测试- MBTI】 即可使用**

### 核心功能

- **多种心理测试**：包含MBTI性格测试、PHQ-9抑郁筛查、SDS抑郁自评量表、SAS焦虑自评量表、DASS-21情绪量表、SCL-90症状自评量表等20+种测试
- **个性化报告**：测试完成后生成详细的分析报告，包含深度分析、成长建议、职业建议等内容
- **数据本地存储**：所有测试数据仅保存在用户本地设备，保护用户隐私
- **分享功能**：支持分享测试结果到微信好友和朋友圈

## 主要功能介绍

### 1. 心理测试模块

- 支持多种测试类型：性格测试、心理健康测试、职业测评等
- 自动跳转下一题，提升答题体验
- 支持进度保存和继续答题

### 2. 结果计算模块

- 根据用户答题结果计算测试分数
- 匹配对应的类型名称和分析报告
- 分析报告给出了心理状态、未来规划、职业规划、恋爱建议等
- 支持多种计分方式（直接计分、反向计分等）

### 3. 报告模块

- 生成详细的测试报告
- 包含深度分析、成长建议等内容
- 支持报告保存和查看历史报告

### 4. 数据管理模块

- 本地存储测试历史和报告
- 支持收藏喜欢的测试
- 测试数据导出功能

## 开发过程中遇到的问题及解决方案

### 问题1：测试类型名称显示错误

**问题描述**：霍兰德职业测试、艾森克人格问卷等测试在报告列表和详情页显示类型代码（如字母"E"）而不是中文名称。

**解决方案**：

- 创建了统一的类型名称映射模块 `type-name-mapper.js`
- 根据测试类型配置不同的映射规则（基于类型代码或分数范围）
- 在报告保存和显示时自动转换类型名称

### 问题2：测试结果计算错误

**问题描述**：卡特尔16PF、霍兰德职业测试等测试结果计算不正确，所有结果都显示相同。

**解决方案**：

- 发现计算函数中使用了错误的 `testId`（如16PF使用了testId=16而不是正确的9）
- 修正了所有测试类型的 `testId` 映射
- 确保每个测试使用正确的题目数据进行计算

### 问题3：分享功能无法使用

**问题描述**：点击朋友圈分享按钮无法触发分享。

**解决方案**：

- 了解到微信小程序的朋友圈分享必须通过右上角菜单触发
- 修改分享按钮逻辑，引导用户使用正确的分享方式
- 完善了 `onShareAppMessage` 和 `onShareTimeline` 生命周期方法

### 问题4：重复保存报告

**问题描述**：用户多次点击保存按钮会保存多份相同的报告。

**解决方案**：

- 在页面中添加 `hasSaved` 状态标记
- 保存成功后设置标记，重复点击时显示提示
- 不同测试会话可以保存多份报告

## 项目报告

### 1. 项目背景

**随着社会竞争的加剧，大专生面临着学业、就业、人际关系等多方面的压力，心理健康问题日益突出。本项目旨在为大专生提供一个便捷、专业的心理健康测评工具，帮助他们了解自己的心理状态。**

### 2. 目标用户

- **大专院校学生**：主要目标用户，帮助他们进行心理健康自测
- **心理咨询师**：可作为辅助工具了解来访者基本情况
- **辅导员**：可以及时了解学生的心理状态
- **普通用户**：对心理测试感兴趣的人群

### 3. 功能特点

| 特点   | 说明              |
| ---- | --------------- |
| 专业可靠 | 基于经典心理学量表开发     |
| 隐私保护 | 数据仅保存在本地，不上传服务器 |
| 操作便捷 | 简洁的界面设计，易于使用    |
| 内容丰富 | 涵盖多种测试类型，满足不同需求 |

### 4. 发展方向

- 增加更多专业心理测试
- 开发心理咨询预约功能
- 添加心理健康资讯板块
- 支持多语言版本
